    🎹 Synths (4 ondas crudas)
    
        
    
          Pulso NO tiene samples reales de guitarras/violines/etc.
          Todo es síntesis con estas 4 ondas. Para sonidos específicos usa los
          presets abajo, o exporta MIDI y carga en un DAW con tus VSTs.
        
    
        
    
          .synth('sine')
          Suave / pura.
          .synth('square')
          8-bit retro.
          .synth('sawtooth')
          Brillante.
          .synth('triangle')
          Dulce.

    🎮 Compatibilidad GBA · Pokemon decomp · DAWs
    
        
    
          El botón Export MIDI genera un Standard MIDI File
          formato 1. Verificado con file como
          "Standard MIDI data (format 1)". Compatible con:
        
    
        
      
          
    • pokeemerald / pokeruby / pokefirered (decomps):
      
            convierte con mid2agb (parte de devkitARM):
      
            mid2agb -G127 tu.mid 60 0 tu.s
            Copia tu.s a sound/songs/ del decomp y
            registra la entrada en songs.json + sound/song_table.inc.
          
      • 
          
    • MaxMod / Krawall (motores GBA tracker):
      
            abre el MIDI en OpenMPT o MilkyTracker → guarda como .xm /
            .s3m → compila con mmutil.
          
      • 
          
    • DAWs (Ableton, Logic, Reaper, FL, MuseScore, GarageBand):
      
            drag-and-drop directo del .mid. Cada pista de Pulso se
            mapea a una pista MIDI.
          
      • 
        
    
        
    
          Limitaciones para Pokemon decomp:
          el MIDI sale en canal 0 sin program changes — vas a tener que asignar
          el voicegroup/instrumento en el .s después
          de convertir. La percusión usa General MIDI standard (kick=36, snare=38,
          hat=42, clap=39), que coincide con el bank por defecto del M4A.
